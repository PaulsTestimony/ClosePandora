# Close Pandora

**Close Pandora** is a secure, minimal browser app for Android. It is designed to:

- Block all web access except for a password-protected list of trusted domains
- Intercept browser intents and handle email or banking links securely
- Prevent general-purpose browsing
- Provide a clean, dark UI with custom branding

### Features

- ✅ WebView with strict domain whitelisting
- 🔐 Admin password protection
- 🚫 Blocks all other browser apps
- 📬 Opens only trusted links (e.g., from email or work apps)
- 🌙 Dark theme with locked Pandora’s Box branding

### Build Instructions

This project builds automatically with GitHub Actions.
To build manually:
1. Clone the repo
2. Open in Android Studio
3. Click **Build → Build APK**
